# Megalodon

UCI chess engine.

[Version 0.0.3][latest] was recently released!

## To run with `cmake`

1. Clone or download the repository,
2. Create `build` directory
3. `cd` to `build/`
4. Create `Makefile`

``` bash
mkdir build
cd build/
cmake ..
```

## The Name

The name megalodon comes from an extinct species of shark that lived millions of years ago.
